# 해빗 뽀모도로

앱인토스 제출을 목표로 만든 습관 트래커 + 뽀모도로 타이머. React + Vite + TypeScript + Tailwind CSS.

## 화면 구성 (하단 탭)
- **할일 등록**: 알람식 시/분 피커(최소 10분)로 집중 시간 설정, 우선순위(긴급/보통/낮음) 지정 후 타이머 시작
- **뽀모도로 타이머**: 일시정지/재개, 시간 남았을 때 종료 시 중립적인 확인 팝업(다크패턴 방지 고려)
- **완료 항목**: 최신순/오래된순 정렬, 항목별 상세(집중 시간·일시정지 횟수) 아코디언, 개별 삭제
- **월간 트래커**: 캘린더에 우선순위별 색상 점 표시(최대 5개), 이번 달 요약 통계

## 현재 상태
지금은 **브라우저 로컬 저장(localStorage)만 지원**해서 기기 간 동기화가 안 돼요.
앱인토스는 로그인을 토스 로그인만 허용하므로, 실제 배포 전 토스 로그인 + Supabase(RLS로 사용자별 데이터 격리)를 붙일 예정입니다.

## 로컬 실행
```bash
npm install
npm run dev
```
